# 開發伺服器啟動說明

## 問題解決

**問題**：端口 5173 被占用  
**解決方案**：已修改配置使用端口 **3000**

## 訪問網站

開發伺服器現在運行在：
```
http://localhost:3000
```

瀏覽器應該已經自動開啟。如果沒有，請手動訪問上述網址。

## 手動啟動步驟

如果開發伺服器沒有自動啟動，請執行：

```powershell
cd c:\Web\drservWeb
npm run dev
```

## 檢查伺服器狀態

如果無法訪問，請檢查：

1. **確認 Node.js 進程正在運行**
   ```powershell
   Get-Process | Where-Object {$_.ProcessName -eq "node"}
   ```

2. **檢查端口 3000 是否被占用**
   ```powershell
   Get-NetTCPConnection -LocalPort 3000 -ErrorAction SilentlyContinue
   ```

3. **查看終端機輸出**
   - 應該會顯示：`Local: http://localhost:3000/`

## 停止伺服器

在運行 `npm run dev` 的終端機視窗中按 `Ctrl + C`

## 如果仍有問題

1. 關閉所有 Node.js 進程
2. 重新執行 `npm run dev`
3. 檢查是否有錯誤訊息
