# 🚀 快速安裝 Node.js 並測試網站

## ⚡ 快速步驟

### 1️⃣ 下載並安裝 Node.js

**直接下載連結：**
- **Windows 64-bit**: https://nodejs.org/dist/v20.10.0/node-v20.10.0-x64.msi
- **官方網站**: https://nodejs.org/zh-tw/ （選擇 LTS 版本）

**安裝步驟：**
1. 下載 `.msi` 檔案
2. 雙擊執行安裝程式
3. 全部選擇「下一步」（使用預設選項）
4. ✅ **重要**：確保勾選「Add to PATH」
5. 完成安裝

### 2️⃣ 重新開啟終端機

**必須重新開啟 PowerShell 或命令提示字元**，讓環境變數生效。

### 3️⃣ 驗證安裝

在新的終端機中執行：

```powershell
node --version
npm --version
```

應該會顯示版本號，例如：
```
v20.10.0
10.2.3
```

### 4️⃣ 安裝專案依賴

在專案目錄 `c:\Web\drservWeb` 執行：

```powershell
npm install
```

這會下載所有必要的套件（約 2-3 分鐘）

### 5️⃣ 啟動開發伺服器

```powershell
npm run dev
```

成功後會看到：
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
```

瀏覽器會自動開啟，您就可以看到完整的網站了！🎉

## 📋 目前可用的預覽版本

我已經創建了一個 **`preview.html`** 檔案，您可以直接在瀏覽器中打開查看基本樣式：

1. 在檔案總管中找到 `c:\Web\drservWeb\preview.html`
2. 雙擊開啟（會用預設瀏覽器開啟）
3. 這是一個簡化版本，展示網站的基本結構和樣式

**注意**：預覽版本不包含完整的 React 功能和動畫效果，僅供快速查看樣式。
