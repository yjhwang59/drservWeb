## 管理自動啟動

### 查看啟動項目

1. 按 `Win + R`
2. 輸入：`shell:startup`
3. 按 Enter
4. 可以看到 `DrServ-DevServer.lnk` 捷徑

### 停用自動啟動

**方法一：刪除捷徑**
- 在啟動資料夾中刪除 `DrServ-DevServer.lnk`

**方法二：重新命名**
- 將捷徑重新命名為 `DrServ-DevServer.lnk.bak`（停用）
- 需要時再改回 `.lnk`（啟用）

### 手動啟動（不使用自動啟動）

直接執行：
```powershell
cd C:\Web\drservWeb
.\start-dev-server.ps1
```

或雙擊：
```
C:\Web\drservWeb\start-dev-server.bat
```

## 注意事項

1. **防火牆設定**
   - 首次啟動時，Windows 可能會詢問是否允許 Node.js 通過防火牆
   - 請選擇「允許存取」

2. **開機時間**
   - 開發伺服器會在開機後自動啟動
   - 可能需要等待幾秒鐘讓 Node.js 和依賴載入完成

3. **停止伺服器**
   - 在工作管理員中找到 Node.js 進程並結束
   - 或在 PowerShell 視窗中按 `Ctrl + C`

4. **查看日誌**
   - 伺服器輸出會顯示在自動開啟的 PowerShell 視窗中
   - 可以查看是否有錯誤訊息

## 疑難排解

### 如果自動啟動失敗

1. **檢查腳本路徑**
   - 確認 `C:\Web\drservWeb\start-dev-server.bat` 存在

2. **檢查 Node.js**
   - 開啟新的 PowerShell，執行 `node --version`
   - 如果沒有顯示版本，需要重新安裝 Node.js

3. **檢查權限**
   - 確認啟動資料夾有寫入權限
   - 可能需要以系統管理員身份執行設定腳本

4. **查看事件檢視器**
   - 開啟「事件檢視器」→「Windows 記錄」→「應用程式」
   - 查看是否有相關錯誤訊息
