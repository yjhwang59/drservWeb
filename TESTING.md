# 測試指南

## 方式一：本地開發測試（推薦）

### 步驟 1：安裝 Node.js

1. 前往 [Node.js 官網](https://nodejs.org/) 下載並安裝 LTS 版本（建議 18.x 或更高）
2. 安裝完成後，重新開啟終端機
3. 驗證安裝：
   ```bash
   node --version
   npm --version
   ```

### 步驟 2：安裝專案依賴

在專案目錄下執行：

```bash
npm install
```

### 步驟 3：啟動開發伺服器

```bash
npm run dev
```

開發伺服器將在 `http://localhost:5173` 啟動，瀏覽器會自動開啟。

### 步驟 4：測試功能

- ✅ 檢查所有頁面區塊是否正常顯示
- ✅ 測試導航列平滑滾動功能
- ✅ 測試響應式設計（調整瀏覽器視窗大小）
- ✅ 測試聯絡表單驗證
- ✅ 檢查動畫效果是否流暢

## 方式二：構建後預覽

### 構建生產版本

```bash
npm run build
```

### 預覽構建結果

```bash
npm run preview
```

預覽伺服器將在 `http://localhost:4173` 啟動。

## 常見問題

### Q: npm install 失敗？

**A:** 檢查：
1. Node.js 是否正確安裝
2. 網路連線是否正常
3. 嘗試清除快取：`npm cache clean --force`

### Q: 開發伺服器無法啟動？

**A:** 檢查：
1. 端口 5173 是否被占用
2. 嘗試使用其他端口：`npm run dev -- --port 3000`

### Q: 樣式沒有載入？

**A:** 確認：
1. `tailwind.config.js` 配置正確
2. `src/index.css` 已正確導入 Tailwind 指令
3. 重新啟動開發伺服器
